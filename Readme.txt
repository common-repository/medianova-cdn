A Content Delivery Network (CDN) is a geographically distrubuted group of servers which run together handing over static contents.
 
== Description ==
 
A CDN is a server network system that enables your users to access your website faster and reach your site by serving them from the servers located near them.

== What it Does ? ==
Medianova CDN plugin is created to easily modify the links of your static content (css, js, images) with cdn url.

== Features ==
* Changes the paths of your files to CDN url
* You can declare the included folders.
* You can declare invalid folders and file extension.


== Manuel Installation ==
 
This section describes how to install the plugin and get it working.
 
1. Download the plugin as medianova-cdn.zip file and extract to your pc.
2. Upload `medianova-cdn` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
